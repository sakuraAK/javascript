# Simple Todo App

A lightweight, single-page todo list application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Add Tasks** - Create new todo items with a simple input field
- **Mark Complete** - Check off tasks as you complete them
- **Delete Tasks** - Remove individual tasks or clear all completed items at once
- **Filter Options** - View all tasks, only active, or only completed tasks
- **Persistent Storage** - Todos automatically save to browser's local storage
- **Task Counter** - See how many active tasks remain
- **Responsive Design** - Works seamlessly on desktop and mobile devices

## How to Use

1. Open `index.html` in your browser
2. Type a task in the input field and click "Add" (or press Enter)
3. Check the checkbox to mark tasks as complete
4. Click "Delete" to remove individual tasks
5. Use filter buttons to view different task categories
6. Click "Clear Completed" to remove all finished tasks

## Technical Details

- **Language**: Vanilla JavaScript (ES6)
- **Storage**: Browser localStorage API
- **Styling**: Modern CSS with gradients and animations
- **No Dependencies**: Pure HTML, CSS, and JavaScript

All data persists locally in your browser - no server required!
